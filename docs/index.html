Install in under a minute

The one-liner avoids browser quarantine entirely, so Gatekeeper never blocks it — on any macOS version.

  1. Open Terminal (⌘-Space, type “Terminal”), paste the command above, press Return. It downloads the latest release and launches R9-EQ.
  2. Approve System Audio Recording when macOS asks (System Settings → Privacy & Security). Required — without it the EQ is silent.
  3. Click the R9 icon in your menu bar and press the power button. Done — every sound on your Mac now runs through the EQ.
